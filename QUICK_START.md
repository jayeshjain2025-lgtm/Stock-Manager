# Stock Manager - Quick Start Guide

## Overview
Stock Manager is a comprehensive inventory management system designed for supermarkets and retail chains. It features role-based access with separate interfaces for Owners and Cashiers, making stock management efficient and scalable.

## Getting Started

### 1. Launch the Application
- Open `index.html` in your web browser
- You'll see the role selection page

### 2. Choose Your Role

#### 👑 Owner Dashboard
**Perfect for:**
- Supermarket chain owners
- Multi-location retail managers
- Business owners overseeing multiple shops

**Key Features:**
- **Multi-Shop Management**: View and manage inventory across all locations
- **Profit Analytics**: Track profits and performance per shop
- **Bulk Ordering**: Place orders for multiple products at once
- **Expired Item Management**: Remove expired products system-wide
- **Comprehensive Reports**: Generate detailed inventory and profit reports

#### 🛒 Cashier Terminal
**Perfect for:**
- Point-of-sale staff
- Checkout operators
- Sales associates

**Key Features:**
- **Quick Product Search**: Scan barcodes or search by name
- **Real-time Stock Updates**: Automatic inventory reduction on sales
- **Receipt Generation**: Professional receipts with itemized billing
- **Stock Level Visibility**: See available quantities during checkout
- **Simple Interface**: Focused on speed and efficiency

## System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- JavaScript enabled
- No internet connection required (runs locally)

## Navigation

### Owner Interface
1. **Dashboard**: Overview of all shops and key metrics
2. **Inventory**: Detailed product management across locations
3. **Orders**: Bulk ordering and supplier management
4. **Analytics**: Profit reports and performance insights
5. **Settings**: System configuration and user management

### Cashier Interface
1. **Checkout**: Main sales interface with product search
2. **View Stock**: Quick inventory lookup
3. **Receipts**: Transaction history and receipt printing

## Key Workflows

### For Owners
1. **Daily Operations**:
   - Check alerts for low stock and expired items
   - Review profit summaries across all shops
   - Place orders for products running low

2. **Weekly Tasks**:
   - Generate performance reports
   - Remove expired items
   - Analyze sales trends

### For Cashiers
1. **During Sales**:
   - Search for products by name or barcode
   - Add items to cart
   - Process payment and generate receipt
   - Stock levels update automatically

2. **End of Day**:
   - Review transaction history
   - Check remaining stock levels

## Data Management
- All data is stored locally in the browser
- No external database required
- Data persists between sessions
- Export/import capabilities available

## Security Features
- Role-based access control
- No sensitive data stored
- Local-only operation
- No internet connectivity required
